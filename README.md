# 🧠 About Project

**Employee Management System** is an interactive, console-based Python application tailored to demonstrate the core architecture of **Object-Oriented Programming (OOP)**. 

The project models real-world corporate entities into distinct classes (`Person`, `Employee`, and `Manager`), managing dynamic properties such as credentials, financial compensation blueprints, and operational department groupings. It offers an interactive menu engine allowing users to instantiate customized tracking instances natively during runtime.

# ✨ Features

## 🏢 Class Archetype Blueprinting
* ⭐ **Dynamic Entity Generation:** Instantly spin up dedicated objects for generic People, specialized Employees, or departmental Managers.
* ⭐ **Granular Information Parsing:** Capture real-time user arguments including unique corporate IDs, salary baselines, operational fields, and ages.
* ⭐ **Runtime Entity Persistence:** Stores instantiated object states locally in memory, allowing users to call up structural profile states on demand via nested navigation matrices.

# ▶️ How It Works

### Step 1

Launch the console system interface.

### Step 2

Select choices 1 through 3 to create data models for Persons, Employees, or Managers.

### Step 3

Navigate to choice 4 (`Show Details`) to inspect the actively running instances inside memory.

### Step 4

Safely exit the control terminal runtime using choice 5.
